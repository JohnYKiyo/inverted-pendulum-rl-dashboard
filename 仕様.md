# 🚀 Mission: Inverted Pendulum Reinforcement Learning Dashboard

ブラウザ上で「倒立振り子」の強化学習（DQN: Deep Q-Network）を実行し、その学習過程をリアルタイムで可視化するモダンなNext.jsアプリを構築してください。

## 🛠 技術スタック

- **Framework:** Next.js (App Router), TypeScript
- **Styling:** Tailwind CSS, shadcn/ui (風のクリーンなデザイン)
- **AI/ML:** TensorFlow.js (@tensorflow/tfjs)
- **Charts:** Recharts (報酬の推移を可視化)
- **Icons:** Lucide React

## 💡 アプリの構成と要件

### 1. 物理シミュレーション (Physics Engine)

- HTML5 Canvasを使用して、カートと振り子（棒）を描画。
- 物理演算（重力、摩擦、カートの加速度）をスクラッチで実装。

### 2. 強化学習ロジック (DQN Engine)

- TensorFlow.jsを使用して、DQNエージェントを構築。
- **State:** [カート位置, カート速度, 棒の角度, 棒の角速度]
- **Action:** [左へ押す, 右へ押す]
- **Reward:** 棒が垂直に近いほど加点。倒れたらエピソード終了。
- 学習の数式: $Q(s, a) \leftarrow Q(s, a) + \alpha [r + \gamma \max_{a'} Q(s', a') - Q(s, a)]$

### 3. UI/UX デザイン

- **研究者用ダッシュボード風:** ダークモード、グリッドレイアウトを採用。
- **コントロールパネル:** 学習の「Start / Stop / Reset」ボタン。
- **リアルタイム統計:** 現在のエピソード数、ステップ数、累積報酬の表示。
- **チャート:** Rechartsによる報酬の推移グラフ。学習が進むにつれ、右肩上がりになる様子を見せる。

## 🎬 実装の進め方

1. `components/` フォルダに物理エンジン、AIロジック、UIコンポーネントを適切に分割して作成。
2. `page.tsx` でそれらを統合。
3. デモ動画として映えるよう、各ファイルのコードは読みやすく、コメントを充実させてください。
